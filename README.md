# Project EHR

## Overview
Project EHR is an electronic health record management system that provides an efficient way to manage patient information, appointments, and medical history. It aims to improve healthcare quality and streamline administration.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/musical3006swar/ProjectEHR.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ProjectEHR
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000` to access the application.
